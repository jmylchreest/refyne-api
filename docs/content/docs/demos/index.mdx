## Building Your Own

These demos are open source and available in the [Refyne repository](https://github.com/jmylchreest/refyne-api). Use them as starting points for your own integrations.

Each demo includes:
- Complete source code
- Schema definitions for extraction
- API integration patterns
- Error handling examples
