## Get Webhook

`GET /api/v1/webhooks/{id}`

Retrieve details of a specific webhook.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Webhook ID |

### Response

```json
{
  "id": "01HXYZ...",
  "name": "Production Notifications",
  "url": "https://your-server.com/webhook",
  "has_secret": true,
  "events": ["job.completed", "job.failed"],
  "headers": [
    {"name": "X-Custom-Header", "value": "custom-value"}
  ],
  "is_active": true,
  "created_at": "2024-01-15T10:30:00Z",
  "updated_at": "2024-01-15T10:30:00Z"
}
```

## Update Webhook

`PUT /api/v1/webhooks/{id}`

Update an existing webhook configuration.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Webhook ID |

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Unique name for this webhook |
| `url` | string | Yes | Webhook URL |
| `secret` | string | No | New secret (leave empty to keep current) |
| `events` | string[] | No | Event types to subscribe to |
| `headers` | object[] | No | Custom headers to include |
| `is_active` | boolean | Yes | Whether the webhook is active |

### Example Request

```bash
curl -X PUT https://api.refyne.uk/api/v1/webhooks/01HXYZ... \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Production Notifications",
    "url": "https://your-server.com/new-webhook",
    "events": ["*"],
    "is_active": true
  }'
```

### Response

Returns the updated webhook object.

## Delete Webhook

`DELETE /api/v1/webhooks/{id}`

Delete a webhook. This action cannot be undone.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Webhook ID |

### Example Request

```bash
curl -X DELETE https://api.refyne.uk/api/v1/webhooks/01HXYZ... \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response

```json
{
  "success": true
}
```

### Error Responses

| Status | Description |
|--------|-------------|
| 401 | Unauthorized |
| 403 | Forbidden (webhook belongs to another user) |
| 404 | Webhook not found |
