## List Webhook Deliveries

`GET /api/v1/webhooks/{id}/deliveries`

Returns recent delivery attempts for a specific webhook.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Webhook ID |

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | integer | 50 | Maximum deliveries to return (1-100) |
| `offset` | integer | 0 | Number of deliveries to skip |

### Example Request

```bash
curl "https://api.refyne.uk/api/v1/webhooks/01HXYZ.../deliveries?limit=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json
      "max_attempts": 3,
      "created_at": "2024-01-15T10:30:00Z",
      "delivered_at": "2024-01-15T10:30:00Z"
    },
    {
      "id": "01HGHI...",
      "webhook_id": "01HXYZ...",
      "job_id": "01HJKL...",
      "event_type": "job.failed",

```

### Delivery Status

| Status | Description |
|--------|-------------|
| `pending` | Delivery not yet attempted |
| `success` | Successfully delivered (2xx response) |
| `failed` | All retry attempts exhausted |
| `retrying` | Failed, scheduled for retry |

## List Job Webhook Deliveries

`GET /api/v1/jobs/{id}/webhook-deliveries`

Returns all webhook deliveries for a specific job, including both persistent and ephemeral webhooks.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Job ID |

### Example Request

```bash
curl "https://api.refyne.uk/api/v1/jobs/01HDEF.../webhook-deliveries" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response

```json
{
  "job_id": "01HDEF...",
  "deliveries": [
    {
      "id": "01HABC...",
      "webhook_id": "01HXYZ...",
      "job_id": "01HDEF...",
      "event_type": "job.completed",
      "url": "https://your-server.com/webhook",
      "status_code": 200,
      "response_time_ms": 150,
      "status": "success",
      "attempt_number": 1,
      "max_attempts": 3,
      "created_at": "2024-01-15T10:30:00Z",
      "delivered_at": "2024-01-15T10:30:00Z"
    },
    {
      "id": "01HMNO...",
      "webhook_id": null,
      "job_id": "01HDEF...",
      "event_type": "job.completed",
      "url": "https://other-service.com/callback",
      "status_code": 200,
      "response_time_ms": 200,
      "status": "success",
      "attempt_number": 1,
      "max_attempts": 3,
      "created_at": "2024-01-15T10:30:00Z",
      "delivered_at": "2024-01-15T10:30:00Z"
    }
  ]
}
```

Note: Deliveries with `webhook_id: null` are from ephemeral webhooks specified inline with the job request.
