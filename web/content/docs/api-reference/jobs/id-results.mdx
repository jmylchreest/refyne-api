## Get Job Results

`GET /api/v1/jobs/{id}/results`

Returns the extraction results from a completed crawl job.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Job ID |

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `merge` | boolean | `false` | Merge all results into a single object with deduplicated items |
| `format` | string | `json` | Output format: `json`, `jsonl`, or `yaml` |

### Example Request

```bash
# JSON format (default)
curl "https://api.refyne.uk/api/v1/jobs/01HXYZ.../results" \
  -H "Authorization: Bearer YOUR_API_KEY"

# JSONL format (one result per line)
curl "https://api.refyne.uk/api/v1/jobs/01HXYZ.../results?format=jsonl" \
  -H "Authorization: Bearer YOUR_API_KEY"

# YAML format
curl "https://api.refyne.uk/api/v1/jobs/01HXYZ.../results?format=yaml" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Merged results as YAML
curl "https://api.refyne.uk/api/v1/jobs/01HXYZ.../results?merge=true&format=yaml" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response (JSON, merge=false)

```json
{
  "job_id": "01HXYZ...",
  "status": "completed",
  "page_count": 3,
  "results": [
    {
      "id": "01HABC...",
      "url": "https://example.com/page1",
      "data": {"title": "Page 1", "products": [...]}
    },
    {
      "id": "01HDEF...",
      "url": "https://example.com/page2",
      "data": {"title": "Page 2", "products": [...]}
    }
  ]
}
```

### Response (JSON, merge=true)

When `merge=true`, all results are merged intelligently:
- Arrays are concatenated and deduplicated (by `url` field if present)
- Scalar values take the first non-null value

```json
{
  "job_id": "01HXYZ...",
  "status": "completed",
  "page_count": 3,
  "merged": {
    "products": [
      {"name": "Product A", "price": 29.99, "url": "..."},
      {"name": "Product B", "price": 49.99, "url": "..."}
    ],
    "site_name": "Example Store"
  }
}
```

### Response (JSONL)

With `format=jsonl`, returns one result per line (newline-delimited JSON):

```
{"url":"https://example.com/page1","data":{"title":"Page 1","products":[...]}}
{"url":"https://example.com/page2","data":{"title":"Page 2","products":[...]}}
```

### Response (YAML)

With `format=yaml`, returns YAML-formatted output:

```yaml
- url: https://example.com/page1
  data:
    title: Page 1
    products:
      - name: Product A
        price: 29.99
- url: https://example.com/page2
  data:
    title: Page 2
    products:
      - name: Product B
        price: 49.99
```

### Content Types

| Format | Content-Type |
|--------|--------------|
| `json` | `application/json` |
| `jsonl` | `application/x-ndjson` |
| `yaml` | `application/yaml` |
